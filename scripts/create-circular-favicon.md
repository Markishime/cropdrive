# Creating Circular Favicon

To create circular favicon files for CropDrive:

1. Take the CropDrive logo (`/public/images/CropDrive.png`)
2. Create a circular version with:
   - Transparent circular background
   - Logo centered
   - Proper padding around the logo
3. Export as:
   - `favicon-16x16.png` (16x16 pixels)
   - `favicon-32x32.png` (32x32 pixels)
   - `favicon.ico` (multi-size ICO file)
   - `apple-touch-icon.png` (180x180 pixels)

The favicon files should be placed in `/public/` directory.

Tools you can use:
- Photoshop/GIMP with circular mask
- Online favicon generators that support circular icons
- Image editing software with circular crop feature
